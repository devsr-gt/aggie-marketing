# Assets — Paul Schat's Bakery

## Folder Structure

- `raw/` — original unedited footage and photos straight from camera/phone
- `edited/` — final export files ready to post
- `posted/` — archive of everything that has gone live (rename with date posted)

## Naming Convention
Use consistent naming so files are easy to find:

```
YYYY-MM-DD_platform_content-type_description
```

**Example:**
```
2026-04-10_IG_reel_spring-menu-launch.mp4
2026-04-10_FB_photo_team-spotlight-maria.jpg
```
