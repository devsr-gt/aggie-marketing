# Category Playbooks

Each folder contains a content playbook for that business type — hooks that convert, CTAs that work, what metrics to watch, and what kinds of content perform best in that category.

Use these playbooks when:
- Onboarding a new client in that category
- Planning a monthly content strategy
- Brainstorming hooks or CTAs
- Training on a new niche
