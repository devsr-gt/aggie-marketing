# Processes — Internal SOPs

Standard operating procedures for running Aggie Marketing efficiently.

---

## New Client Onboarding
1. Send onboarding questionnaire (`_templates/onboarding-questionnaire.md`)
2. Conduct discovery call (record with permission)
3. Duplicate `clients/_template/` → rename to client name
4. Fill out `00-onboarding/` and `01-brand/` docs within 48 hours of call
5. Build first strategy doc (`02-strategy/`) before any content is created
6. Send contract and collect first payment
7. Schedule first content shoot within 2 weeks of signing

## Monthly Content Workflow
1. **Week 1:** Plan — fill out content calendar for the month based on strategy
2. **Week 2:** Shoot — capture all raw footage for the month in one or two sessions
3. **Week 3:** Edit — process all footage, write captions, prepare for scheduling
4. **Week 4:** Schedule — queue all content, review analytics from prior month, send report

## Shoot Day Checklist
- [ ] Review content calendar before arriving
- [ ] Charged phone/camera + extra batteries
- [ ] Shot list printed or on phone
- [ ] B-roll list (products, space, details)
- [ ] Client briefed on what to wear and who should be on camera
- [ ] Capture at least 2–3x more footage than needed
- [ ] Transfer raw files to `04-assets/raw/` same day

## Monthly Reporting
- Pull analytics from native platform insights (Instagram, Facebook, TikTok)
- Fill out `05-analytics/README.md`
- Write client-facing summary using `_templates/monthly-report-template.md`
- Send report within the first 5 days of the new month
- Schedule 15-minute review call with client
